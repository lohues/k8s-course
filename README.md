# Kubernetes Crash Course

A comprehensive, hands-on journey from zero to production-ready Kubernetes across 5 progressive stages.

**Live site:** <https://k8s-course.louisdav.id/>

## Stages

| Stage | Title | Topics |
|-------|-------|--------|
| 1 | Prerequisites | Linux, Docker, Networking, YAML & CLI fundamentals |
| 2 | Core Concepts | Pods, Deployments, Services, kubectl & ConfigMaps |
| 3 | Intermediate | Scheduling, Storage, Ingress, Workloads, RBAC & Probes |
| 4 | Real-World Ops | Helm, CI/CD, ArgoCD, Monitoring & Auto-Scaling |
| 5 | Advanced | kubeadm, CRDs, Istio, Security, etcd & Multi-Cluster |

Each stage includes theory sections and a hands-on lab with YAML manifests in the `labs/` directory.

## Running Locally

Open `index.html` in a browser. No build step or server required — the course is a static site.

## PWA Support

The site can be installed as a Progressive Web App on desktop and mobile. On iOS, use Safari's **Add to Home Screen** to get a standalone app experience.
